
# AI Voice Bot 

This repository contains a MVP version version of a voice bot built using Retell AI and WebRTC.The actuall Project I have Integerated with Twillo Phone number and Mobile Number For a Hospital Customer Support to automate their Appointment Booking System 

## Project Description

This is a MVP version of  voice bot that uses Retell AI together with a web call flow built on WebRTC. It connects to the Rittle AI API through an API key and applies prompt engineering to shape the AI responses.

## Demo Notes

- The current repository is a demo deployment only.
- WebRTC is used for web-based voice calling.
- The bot connects to Retell/Rittle AI using an API key.
- Prompt engineering is included to control the listener experience.
- The full production project is integrated with Twilio and deployed to internal tools for end-to-end appointment booking.

## What This Demo Shows

- React-based voice bot UI
- WebRTC browser call support
- Retell AI / Rittle AI API integration
- Prompt engineering for conversational flow
